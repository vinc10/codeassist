How to Use Code Assistant

This tool is designed to help you improve your code with suggestions from a Large Language Model (LLM). Simply paste your code into the "Your Code" text area and click the "Fix Code" button. The AI will analyze your code and provide recommendations for improvements and corrections in the "Recommended Improvements" section. If your code is already in good shape, the AI will let you know that no changes are necessary.

The code assistant was created by Vincent Bohman for Gothenburg University. https://github.com/vinc10/codeassist
